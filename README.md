# School Hub

A focused school landing project with a React client and an Express API server.

## Structure

- `client`: Vite React public school website.
- `server`: Express API serving landing-page content from backend data.

## Commands

```bash
npm install
npm run dev
npm run build
```

The client runs on `http://localhost:5173` and proxies `/api` requests to the server on `http://localhost:4000`.
